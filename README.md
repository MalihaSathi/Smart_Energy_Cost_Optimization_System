# Smart_Energy_Cost_Optimization_System
This program is a Smart Energy Cost Optimization System developed in the C programming language that helps users select the most important electrical appliances within a limited power capacity. The user first enters the number of appliances along with details such as appliance name, power consumption, priority level, and usage time. The program then asks for the maximum allowed power limit. Using the Dynamic Programming approach based on the Knapsack algorithm, the program determines the best combination of appliances that provides the highest total priority without exceeding the power limit. After selecting the optimal appliances, the program calculates the total power consumption, total usage time, energy usage in kilowatt-hours (kWh), and the estimated electricity cost based on a fixed electricity rate. Finally, it displays the selected appliances and the complete energy optimization result. The program demonstrates important concepts of C programming including arrays, loops, functions, strings, dynamic programming, optimization techniques, and cost calculation.
